# Утилита аутентификации и авторизации Windows

Приложение предназначено для демонстрации и тестирования нативных механизмов аутентификации операционной системы Windows, получения данных о текущей сессии пользователя и проверки уровней привилегий.

Программа предоставляет интерфейс для взаимодействия с учетными записями безопасности Windows, проверки токенов аутентификации NTLM/Kerberos и определения прав доступа запущенного процесса (UAC).

## Функциональные возможности

* **Анализ сессии Windows:** Автоматическое извлечение данных о текущем пользователе ОС (имя, тип аутентификации NTLM/Kerberos, проверка статусов Anonymous/Guest/System).
* **Доменная аутентификация:** Форма для ручной проверки учетных данных с указанием целевого домена, имени пользователя и пароля.
* **Проверка уровня привилегий:** Модуль для определения наличия прав администратора у текущего контекста запущенного процесса.
* **Журналирование событий:** Встроенная консоль для логирования шагов работы, фиксации попыток входа и системных меток времени.
* **Диагностический вывод:** Отображение подробного контекста безопасности и токенов после прохождения процедуры аутентификации.

## Стек технологий

* **Язык программирования:** C#
* **Инреймворк / UI-библиотека:** WPF


## Структура интерфейса

Архитектура экранных форм оптимизирована для быстрой диагностики и включает в себя три основных блока:

1. **Панель текущего состояния:** Выводит активный контекст пользователя Windows.
2. **Интерактивная зона:** Содержит поля ввода для доменной аутентификации и кнопку проверки прав.
3. **Зона мониторинга:** Отображает лог событий в реальном времени и детальную диагностическую информацию.

## Развертывание и запуск

### Системные требования
* Операционная система Windows (7 / 10 / 11) или Windows Server.
* .NET SDK

### Инструкция по сборке
1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/amadesya/WindowsAuthApp
  ```
2. Перейдите в каталог проекта:
```bash
cd WindowsAuthApp
```

3. Выполните запуск:
``` bash
dotnet run
```
